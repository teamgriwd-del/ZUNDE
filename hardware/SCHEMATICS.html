CN-01 — PFUMA Collar Node
Worn on animal. Solar-charged. Transmits live vitals every 10 s via LoRa.
Circuit Schematic
Device Form Factor
In Proteus: Use Arduino Mega 2560 as proxy for ESP32.
DS18B20 is in the Transducers library.
Replace MPU-6050 with 3× POT-HG (pots A1,A2,A3).
Replace MAX30102 with 1× POT-HG (pot A0).
Add Virtual Terminal on TX0/RX0 — watch live JSON.

BS-01 — PFUMA Base Station
Fixed at farm. Receives all collar LoRa packets. Forwards to PFUMA API via WiFi.
Circuit Schematic
Device Form Factor
In Proteus: Second instance of Arduino Mega 2560.
Connect VT2 (Virtual Terminal) to TX0/RX0.
Two Proteus instances talk via COMPIM virtual serial bridge — one is collar, one is base station.
In KiCad: Clone the ESP32 symbol, reassign GPIO per BS-01 pin table below.
